Prerequisites:
- Node.js installed (version 16 or higher)

How to Run:
Option A: Single Folder Structure
(Just one folder with all files inside)
1. Extract the zip file
2. Open terminal/command prompt
3. Enter command: npm start

Option B: Double Folder Structure
(Has 1 inner and 1 outer folder)
1. Extract the zip file
2. Open terminal/command prompt
3. Navigate to the inner folder containing package.json:
   cd "C:\path\to\inner\folder"
4. Check you're in the right place:
   dir package.json
5. Start the application:
   npm start

Access the Application:
Open browser and go to: http://localhost:3000
